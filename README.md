# João Felipe Silva Freitas - Portfolio Dashboard

> Analista de Cibersegurança | Operações Blue Team

Projeto de portfólio pessoal construído como uma Single Page Application (SPA). A interface simula um ambiente de Security Operations Center (SOC), integrando uma navegação gráfica padrão com um terminal de linha de comando interativo.

🔗 **[Acessar Aplicação em Produção](https://joaofelipe-freitas.github.io/Potif-lio/)**

---

## Tecnologias Utilizadas

- **Frontend:** Next.js, React, TypeScript
- **Estilização:** Tailwind CSS
- **Animações:** Framer Motion
- **Visualização de Dados:** Recharts
- **Gerenciamento de Estado:** Zustand
- **Deploy:** GitHub Actions & Pages

---

## Como rodar localmente

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina (versão 18+ recomendada).

1. Clone o repositório:
```bash
git clone https://github.com/joaofelipe-freitas/Potif-lio.git
```

2. Entre no diretório do projeto:
```bash
cd Potif-lio
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

A aplicação estará acessível em `http://localhost:3000`.
